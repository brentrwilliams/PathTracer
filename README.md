PathTracer
==========

This is a simple Monte Carlo Path Tracer that I have been writing in my free time.

## Requirements
- OSX
- GLFW3
- GLEW
- Xcode

## Running It
./pathTracer [imageWidth imageHeight]

## Keyboard Interaction
| Key              | Action                                       |
|-----------------|------------------------------------------|
| SPACEBAR | Save current render to out.tga |
